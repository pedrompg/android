Objectius del programa: 
    Auxiliar a un alumne alhora de calcular mitjes i guardar informació dels examens de les assignatures matriculades

Funcionalitats:
    El programa fa servir llistes per ensenyar les assignatures matriculades del alumne, si no n'hi ha cap s'ensenya un missatge explicant com accedir a l'ajuda del programa i com afegir assignatures, a la pantalla inicial del programa i un altre llista amb els examens de cada assignatura, les dades es guarden en una base de dades

L'ajuda del programa contè una breu explicació de com treballar amb el programa :
    -Per esborrar assignatures aguanteu el dit a sobre de l'assignatura que desitjis esborrar
    -Els pesos dels examens sempre han de ser un numero entre 1-100
    -El programa accepta multiplicacions alhora de inserir el pes d'un examen, per example si tens un examen que val un 30% del 50% de la nota final, pots introduir 30*50 com el pes de l'examen
    -Per calcular la mitja d'una assignatura, heu d'anar a la llista d'examens de l'assignatura, apretar el botó menu i donar-li a l'opció 'Calcula'
